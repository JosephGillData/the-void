# Phase 2 (Weeks 6–16): Daily Execution Plan

## Summary

- This phase is about **visibility + positioning**, not heavy learning
- You will run **3 parallel tracks daily**:
    - Content (external)
    - Internal signalling (career leverage)
    - Light hands-on (credibility)
- Each day = **60–90 minutes total**, focused and repeatable

---

## Core Weekly Structure

### Every Week Includes:

- 1 LinkedIn post (non-negotiable)
- 2–3 light learning sessions (to fuel content)
- 2 internal actions (signal + positioning)
- 2–3 hands-on sessions (short, practical)

---

# Weekly Rhythm (Repeat for 10 Weeks)

## Monday — Topic + Idea Generation

### Tasks:

- Pick weekly topic (from your 8-post list)
- Ask:
    - “What is the simplest way to explain this?”
- Generate:
    - 3–5 angles for your post

### Output:

- Rough post idea
- 3 bullet insights

---

## Tuesday — Learning + Clarification

### Tasks:

- 1–2 videos on topic
- Ask ChatGPT:
    - “Explain this simply”
    - “What’s the key insight here?”

### Output:

- Clear mental model
- 5 bullet points

---

## Wednesday — Write First Draft

### Tasks:

- Write LinkedIn post

### Structure:

- Hook
- Simple explanation
- Insight
- Question

### Constraint:

- Max 150–250 words
- No overthinking

---

## Thursday — Refine + Sharpen

### Tasks:

- Improve:
    - Hook (make it stronger)
    - Clarity (simplify language)
- Add:
    - One clear opinion

---

## Friday — Publish + Engage

### Tasks:

- Post on LinkedIn
- Reply to comments (if any)

### Optional:

- Message 1–2 colleagues:
    - “I’ve been thinking about AI infrastructure—curious what you think”

---

## Saturday — Hands-on (Light Build)

### Tasks (choose one path and stick to it):

#### Option A:

- Install Docker + run simple container

#### Option B:

- Set up small ML model locally

#### Option C:

- Start basic home lab setup

### Output:

- 2–3 bullet learnings

---

## Sunday — Internal Positioning

### Tasks:

- Do ONE:
    - Draft internal note (short)
    - Message manager about interests
    - Identify relevant internal project
    - Outline presentation idea
